# Student Management System

## Overview

A full-stack Student Management System developed using Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features

* Add Student
* View Students
* Update Student
* Delete Student
* Search Students
* Responsive User Interface

## Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* HTML
* CSS
* JavaScript
* Maven

## Project Structure

* Controller Layer
* Service Layer
* Repository Layer
* Entity Layer
* MySQL Database

## Learning Outcomes

* REST API Development
* CRUD Operations
* Frontend-Backend Integration
* Database Connectivity
* Spring Boot Architecture
